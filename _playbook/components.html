Components

Vets.gov components are page level design elements that help visitors navigate, scan, and understand information. For further guidance visit the U.S. Web design standards.

Navigation

Vets.gov uses a lightweight navigation system.

Breadcrumbs

Breadcrumb links appear in the header of each content page to help a visitor navigate and keep track of their location.
