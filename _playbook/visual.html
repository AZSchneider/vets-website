Images

Image guidelines

Generally, vets.gov avoids the use of imagery, icons and other visual embelishments. If an image is to be used, it should convey content that connect be adquately conveyed by text alone. If an image is used, be sure to caption the meaning of the content through the use of an alt tag to maintain Section 508 accessibility compliance.
